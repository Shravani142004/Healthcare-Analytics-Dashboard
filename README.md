# Healthcare-Analytics-Dashboard

## 📌 Project Overview
This project is an interactive Healthcare Analytics Dashboard developed using Power BI. It provides insights into patient data, revenue, department performance, treatment types, and waiting time through interactive visualizations.

## 🛠️ Tools Used
- Power BI
- Microsoft Excel / CSV
- Data Visualization

## 📊 Dashboard Features
- Total Patients KPI
- Total Revenue KPI
- Average Cost KPI
- Department-wise Patient Distribution
- Treatment Type Analysis
- Average Waiting Time
- Patient Consultation Fee Analysis
- Interactive Date Filter

## 📁 Files Included
- Healthcare_Analytics_Dashboard.pbix
- realistic_healthcare_dataset.csv
- Dashboard Screenshot

## 📈 Key Insights
- Analyzed patient distribution across departments.
- Tracked revenue and consultation fees.
- Visualized treatment trends.
- Monitored average patient waiting time.
